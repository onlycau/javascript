JavaScript 高级程序设计第六章

07/03/2019

  • 函数表达式不同于函数声明，没有名字的函数表达式也叫作匿名函数。
  • 递归函数不要使用函数名来递归的调用自身（非严格模式） 严格模式额外创建一个函数表达式：

    • var factorial2 = (function f(num){ if (num <= 1){ return 1; } else{ return num * f(num - 1); } });
  • 当在函数内部定义了其他函数时，就创建了闭包闭包有权访问包含函数内部的所有变量
  • 闭包可以模仿块级作用域和创建私有变量
